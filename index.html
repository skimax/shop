Shopping List Information

This is a Shopping List application in the market for any User's store.

  • 1. A user can create a new grocery store item by typing their item and either clicking on the Add button or pressing Enter on the Keyboard. The new item will pop-up on the list .
  • 2. User already have 5 items set up for them, you can delete them by creating your own list depending on your store.
  • 3. User can delete items on the list, but needs to check item first and press the cross icon at the corner
  • 4. User must be typing an item before clicking on Add button and if you do a message info will pop-up, and user can also edit item.
